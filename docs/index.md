# Builder Dashboard

Welcome to the Builder Dashboard.  
This dashboard provides real-time project and operations insights for your company.

## What you can view here:

- **Project Budgets vs Actuals**  
  Track estimated vs real costs across all builds.

- **Staff & Contractor Task Assignments**  
  Understand responsibilities, workloads, and upcoming tasks.

- **Vendor Billing Status**  
  Monitor invoicing, payments, and outstanding balances.

- **Progress Tracking & Timelines**  
  View project milestones, delays, and completion estimates.

---

Use the menu on the left to navigate dashboards and project boards.
